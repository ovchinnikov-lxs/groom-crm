# Groom.CRM

## Setup


Click to "Use this template" in github
```bash
# npm
pnpm install
```

## Development Server

Start the development server on http://localhost:3000

```bash
pnpm run dev
```

## Production

Build the application for production:
https://habr.com/ru/articles/438862/


```bash
pnpm run build
```

Locally preview production build:

```bash
pnpm run preview
```
